## Contents

### `runge_kutta_4.py`
A well-commented Python implementation of the **Runge-Kutta 4th Order (RK4)** numerical method for solving ODEs.

**Run it:**
```bash
python runge_kutta_4.py
```

**Example output** solves `dy/dt = y` with `y(0) = 1` (exact solution: `e^t`) and compares RK4 results to the exact answer.
